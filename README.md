# Overview
Swiftで書かれたパストレーシング ベースのレンダラです。githole/edupt をSwiftに移植し機能を追加しました。
現在対応しているアルゴリズムは以下の通りです。
- 重点的サンプリング
- Next Event Estimation

# Usage
Xcodeで最適化を有効にしてビルドし実行すると、レンダリングされた画像がピクチャフォルダに書き出されます。実行完了までには数時間かかるため、ループごとに途中経過が保存される仕様です。
